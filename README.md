# ffi-asm

An alternative to libffi. Generates instructions to do similar capabilities as libffi.

Project Status: `alpha`

### Calling Conventions
- [x] cdecl (c)
- [ ] stdcall (windows only)
- [ ] fastcall

### Supported Platforms
| OS | Architecture |
|----|--------------|
| Linux | x86_64 |
| macOS | x86_64 |
| Windows | x86_64 |
